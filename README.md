# 🏥 Sistema de Clínica - Gestão de Pacientes

Sistema web desenvolvido para gerenciamento de pacientes e orçamentos de serviços clínicos.

## 🚀 Tecnologias utilizadas

- Python
- Flask
- SQLite
- HTML, CSS e JavaScript

## 📌 Funcionalidades

- Cadastro de pacientes
- Listagem de pacientes
- Cadastro de serviços
- Geração de orçamentos
- Estrutura organizada com rotas e models
